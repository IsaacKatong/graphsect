# Agents

Reference [ARCHITECTURE.md](ARCHITECTURE.md) for architecture information.

## Coding Style

- The main language is **TypeScript**.
- All changes should be accompanied by tests. Tests are placed in a `__tests__` folder at the same level as the changed files.
- Types should have fixtures in a `__fixtures__` folder which generate mock data for testing.
- Every newly created folder should include a `README.md` explaining its purpose.
- **React** is used for the UI. Follow standard React conventions for components (`.tsx` files, PascalCase naming) and utility functions (`.ts` files, camelCase naming).
